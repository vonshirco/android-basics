Steps (Using GUI)

> Create a github repository (don't include any readme/licence)
> Copy the github-repo https link provided (.git)
> Go Android Studio Menu bar then VCS > Enable Version Control Integration and Click Ok
> Go to Project the your project and right click on it
> Go to Git > Click on +Add
> Then back to Git > Commit Directory...
> Give a Commit message > Then Commit
> (Code Analysis) Commit
> Then back to Git > Repository > Remotes..
> + add a url (.git) then click Ok > Ok
> The back to Git > Repository > Click Push
> Then Authenticate by entering login credentials or Using Token
> Use Token > Generate > Generate and Copy from Github > paste it in the Android studio
> Click on Login
> Then the Code is pushed to the master branch in the github repository
